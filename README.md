# MCP Explorer

MCP Explorer es una herramienta desarrollada en PHP para recopilar, filtrar y analizar repositorios públicos de GitHub relacionados con el Model Context Protocol (MCP).

## Contexto académico

Este proyecto fue desarrollado en el marco de la **Tesina de Grado** de la Licenciatura en Sistemas de la Universidad Nacional de la Patagonia Austral (UNPA-UARG), bajo la dirección de la Dra. Sandra Isabel Casas. La herramienta fue utilizada para la recolección, depuración y análisis de los repositorios públicos empleados en el estudio.

## Características

- Recolección de repositorios mediante la API de GitHub.
- Filtrado y clasificación de resultados.
- Estadísticas del ecosistema MCP.
- Exportación de datos a Excel.
- Visualización del JSON original de cada repositorio.

## Tecnologías

- PHP
- MySQL
- GitHub REST API
